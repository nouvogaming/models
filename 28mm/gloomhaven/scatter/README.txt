This directory holds a collection of scatter pieces for the
Gloomhaven game. I've curated these from a variety of sources
and made adjustments for my own aesthetic:
- resized/reshaped to be no more than 35-40mm tall
- resized/reshaped to consume most of the hex they reside in
- selected and/or adjusted to allow for models to sit on top

See appropriate src directory for origin and license.
