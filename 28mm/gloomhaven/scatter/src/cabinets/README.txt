# Summary

**Update:** 06/06/2018 Added V2 with more details. Print it on it's back. (Ha, just realized what are supposed to be pouches kind of look like eyes on the doors after printing.) 
_____

Overlay tile for the board game Gloomhaven.  The large version is more like a full size wardrobe.  The small version is probably closer to the correct size.  Since obstacles do not block line of sight in the game I assumed they where below shoulder height for the human scoundrel.

Tile Count:
2x (1 hex) Cabinet

Resolution: 0.01mm
Infill: 20%
