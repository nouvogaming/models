# Summary

This is a overlay tile replacement for the board game Gloomhaven.

Tile Count: 3

Tables and chairs are one piece, no supports needed.

If you don't have a 3D printer this piece is for sale at the Broken Token: https://www.thebrokentoken.com/Table-3

# Print Settings

Printer Brand: Prusa
Printer: i3 MK3
Rafts: Doesn't Matter
Supports: No
Resolution: 0.1mm
Infill: 20%

Notes: 
I have printed this successfully without supports.  There will be a slight about of bridging needed in the center of the table arch and on the very outsides of the table top but it should work.
