# Summary

Overlay tile replacement for the board game Gloomhaven.

Finally, players and monsters can now land on every hex of these boulders.  No more worrying about flying monsters not being able to end their turn on boulders.

Included here are the 1 Hex files, I have the complete set of 7 files including 2 and 3 hex versions for sale on my [Cults page](https://cults3d.com/en/3d-model/game/boulders-for-gloomhaven-flat-top-1-2-3-hex)

Tile Count:
6x (1 hex) Boulder
4x (2 hex) Large Boulders
4x (3 hex) Huge Boulders
