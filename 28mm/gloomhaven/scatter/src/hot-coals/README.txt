# Summary

**UPDATE:** 6/27/2018 Added new versions with smoothed edges and outer ridges restored (had been inverted during original lithophane process).  Also added a thin version at ~2.5mm thick, normal version is ~5mm thick.

Hot Coals overlay tiles for Gloomhaven board game, 1, 2, and 3 Hex.

These are ~4.75mm thick, feel free to scale in Z if you want it a little thinner.

Used http://3dp.rocks/lithophane/ to generate replica of original game artwork from the  Gloomhaven creators pack.  Please only use this model for improving the Gloomhaven board game experience along with any remixes.

Tile Count:
2x (1 hex) Hot Coals
3x (2 hex) Hot Coals
3x (3 hex) Hot Coals
