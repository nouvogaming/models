# Summary

I made this for my daughter, she asked for something to decorate her room. It's a remix from mrvv's version of the Robert Indiana design. Only change I made is that I rounded the vertical corners a bit to make the model more suitable for printing at high speed.

I printed this in 142x142mm size with 0.25mm layer height and 10% infill.
